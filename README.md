# Registration Form - Java Swing

## üìå Description
This project is a desktop application developed in **Java Swing** that implements a **multi-step registration form**. It follows the **MVC (Model-View-Controller)** design pattern and uses **CardLayout** to manage navigation between different screens.

The application is designed as part of the **Graphical User Interfaces** course and focuses on usability, validation, event handling, and visual design in Swing.

---

## ‚ú® Features

- Multi-step registration form with guided navigation
- MVC architecture (Model, View, Controller)
- Navigation between screens using CardLayout
- Email and password validation with regular expressions
- Country and province/state selection
- ComboBox with custom renderer and flag icons
- Keyboard shortcuts:
  - **ENTER** to trigger the default button
  - **Ctrl + E** to open the file chooser
- File selection using JFileChooser
- Export user data to a text file
- Visual enhancements:
  - Gradient header
  - Application logo
  - Scaled images
